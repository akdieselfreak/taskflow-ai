Getting Started

Initial Setup

When you first open TaskFlow AI, you'll go through a simple onboarding process:

  1. Enter Your Name - This personalizes your experience and helps AI identify tasks assigned to you
  2. Choose Your AI Service - Select between:
    • Ollama - For local or self-hosted AI models
    • OpenAI - For GPT models via API
    • Open WebUI - For unified interface with multiple AI models
  3. Configure Your Service - Enter connection details
  4. Select a Model - Choose which AI model to use
  5. Test Connection - Verify everything is working
  6. Smart Name Recognition Setup - AI automatically discovers name variations and nicknames for better task detection
üí° Tip: During setup, TaskFlow AI will automatically discover common nicknames and variations of your name (like "Bob" for "Robert"). This helps the AI recognize tasks assigned to you even when people use different names in emails and messages.

AI Task Extraction

One of TaskFlow AI's most powerful features is extracting multiple tasks from unstructured text.

How It Works

  1. Click ‚ú® AI Extract Task
  2. Paste your text (email, chat, notes, etc.)
  3. AI analyzes and extracts:
    • Multiple distinct tasks (if found)
    • Clear, actionable task titles
    • Brief descriptions
    • Important details, dates, and links
  4. Review and edit each extracted task
  5. Remove unwanted tasks if needed
  6. Save all tasks to your list

Multiple Tasks Support

The AI can now identify and extract multiple tasks from the same text:

  • Automatic Detection - AI identifies multiple action items in complex text
  • Individual Review - Each task can be edited or removed independently
  • Batch Processing - Save all extracted tasks at once
  • Smart Separation - AI distinguishes between different action items

Best Practices

  • Personal Tasks Only - AI identifies tasks assigned to you specifically
  • Review All Results - Check each extracted task before saving
  • Remove Duplicates - Use the √ó button to remove unwanted tasks
  • Provide Detail - More context helps AI extract and separate tasks accurately
üí° Pro Tip: When pasting email threads with multiple people and action items, the AI will extract all tasks relevant to you while ignoring tasks assigned to others. Each task appears as a separate, editable item.

Name Variations Management

One of TaskFlow AI's most powerful features is its ability to recognize you by different names and nicknames:

  • Automatic Discovery - During setup, AI automatically finds common variations of your name
  • Manual Customization - Add, remove, or modify name variations in Settings
  • AI Suggestions - Get AI-powered suggestions for additional name variations
  • Smart Recognition - AI uses all variations when identifying tasks assigned to you
üí° Examples: If your name is "Robert", TaskFlow AI might recognize tasks assigned to "Bob", "Rob", "Bobby", or "Robbie". This works for any name - "Jennifer" becomes "Jen", "Jenny", "Jenn", etc.

System Prompt Customization

The system prompt controls how AI interprets and extracts tasks from your text. This powerful feature allows you to:

  • Fine-tune extraction - Add specific instructions for your workflow
  • Multiple task handling - Control how AI separates and identifies multiple tasks
  • Industry-specific rules - Tailor for legal, medical, or technical contexts
  • Priority detection - Identify urgent vs. normal tasks automatically
  • Custom fields - Extract additional information like deadlines or project codes

‚ö†Ô∏è Important: System Prompt Format

Your custom prompt must instruct the AI to return valid JSON with these required fields:

For single task:

  • title - Task title (string)
  • description - Brief summary (string)
  • notes - Additional details (string)

For multiple tasks:

  • tasks - Array of task objects, each with title, description, and notes

Example Custom Prompts

For Project Managers:
"Extract all tasks and identify project codes. Always include deadlines in the notes field. Prioritize tasks marked as 'urgent' or 'ASAP'. Return JSON with single task format or tasks array for multiple tasks."
For Customer Support:
"Extract all support tickets and customer requests. Include ticket numbers and customer names in notes. Flag escalations. Return JSON with single task format or tasks array for multiple items."

Troubleshooting

Common Issues

AI Connection Failed

  • Verify your API endpoint is correct
  • Check that your API key is valid (OpenAI)
  • Ensure Ollama is running (if using Ollama)
  • Try the connection test in Settings ‚Üí "Reconfigure AI"
  • Check your internet connection

Tasks Not Saving

  • Check browser's local storage isn't full
  • Ensure cookies/storage are enabled
  • Try refreshing the page
  • Clear browser cache and restart

AI Extraction Fails or No Multiple Tasks Found

  • Ensure your text contains actionable content with multiple distinct items
  • Try with more detailed or longer text for better context
  • Check your system prompt in Settings (ensure it supports multiple tasks)
  • Verify your AI service is responding
  • Try pasting entire email threads instead of snippets

Import/Export Issues

  • Export Download Fails - Check browser download settings and popup blockers
  • Import File Not Recognized - Verify file format (.json, .md, .csv) and contents
  • Import Shows Zero Tasks - Check if file contains proper task format or data
  • CSV Import Errors - Ensure CSV has proper headers and is comma-separated
  • Large File Issues - Break large imports into smaller chunks

Name Variations Not Working

  • Check your name variations in Settings are correct and complete
  • Use "Get AI Suggestions" to find additional name variations
  • Ensure variations are separated by commas
  • Try including both formal and informal versions of your name
  • Remember variations are case-insensitive

Multiple Tasks Issues

  • AI Only Finds One Task - Try providing more context or explicitly mention multiple action items
  • Duplicate Tasks - Review extracted tasks and remove duplicates before saving
  • Tasks Missing Context - Include more surrounding text to help AI understand relationships
  • Wrong Tasks Extracted - Verify your name is correct in settings so AI identifies your tasks properly

Data Management Issues

  • Backup Restore Fails - Ensure JSON file is from TaskFlow AI and not corrupted
  • Markdown Import Empty - Check that markdown contains checkbox format tasks (- [ ] or - [x])
  • CSV Format Issues - Verify headers match expected format (Name, Description, Status, etc.)
  • Obsidian Import Problems - Ensure file contains proper YAML frontmatter and task format

Performance Issues

  • Slow Loading - Clear browser cache or try incognito mode
  • High Memory Usage - The app limits displayed tasks to prevent issues
  • Unresponsive UI - Refresh the page to reset the application state
  • Large Export Files - Consider exporting tasks in smaller date ranges

Data & Privacy

  • All tasks are stored locally in your browser
  • No data is sent to external servers (except AI queries)
  • API keys are stored securely in browser storage
  • Clear browser data to reset everything
  • Export tasks by copying from the interface
  • Import/export happens locally - no cloud processing

Getting Help

  • Check Browser Console - Look for error messages in developer tools
  • Try Different Browser - Test in Chrome, Firefox, or Safari
  • Reset Configuration - Use "Reconfigure AI" in Settings to start fresh
  • Clear All Data - Clear browser storage to completely reset the app
  • Test With Sample Data - Try exporting/importing with a small test file first
