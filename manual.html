Getting Started

Account Creation & Authentication

TaskFlow AI requires an account to securely store your data and sync across devices. When you first open TaskFlow AI, you'll go through a comprehensive onboarding process:

  1. Account Setup - Choose between:
    • Sign In - Access your existing account and data
    • Create Account - Set up a new account with secure cloud storage
  2. Account Creation (if creating new account):
    • Choose a unique username
    • Provide email address (optional, for account recovery)
    • Create a secure password (minimum 6 characters)
    • Confirm your password
  3. Automatic Login - After account creation, you're automatically logged in
  4. Enter Your Name - This personalizes your experience and helps AI identify tasks assigned to you
  5. Choose Your AI Service - Select between:
    • Ollama - For local or self-hosted AI models
    • OpenAI - For GPT models via API
    • Open WebUI - For unified interface with multiple AI models
  6. Configure Your Service - Enter connection details (API keys, server URLs)
  7. Select a Model - Choose which AI model to use from available options
  8. Test Connection - Verify everything is working correctly
  9. Smart Name Recognition Setup - AI automatically discovers name variations and nicknames for better task detection
üîê Security Note: Your account uses secure JWT authentication with encrypted password storage. All your tasks, notes, and settings are stored securely in the cloud and synchronized across all your devices.
üí° Tip: During setup, TaskFlow AI will automatically discover common nicknames and variations of your name (like "Bob" for "Robert"). This helps the AI recognize tasks assigned to you even when people use different names in emails and messages.

Once you have an account, you can access TaskFlow AI from any device:

  • Automatic Sync - All data syncs automatically across devices

AI Task Extraction

One of TaskFlow AI's most powerful features is extracting multiple tasks from unstructured text.

How It Works

  1. Click ‚ú® AI Extract Task
  2. Paste your text (email, chat, notes, etc.)
  3. AI analyzes and extracts:
    • Multiple distinct tasks (if found)
    • Clear, actionable task titles
    • Brief descriptions
    • Important details, dates, and links
  4. Review and edit each extracted task
  5. Remove unwanted tasks if needed
  6. Save all tasks to your list

Multiple Tasks Support

The AI can now identify and extract multiple tasks from the same text:

  • Automatic Detection - AI identifies multiple action items in complex text
  • Individual Review - Each task can be edited or removed independently
  • Batch Processing - Save all extracted tasks at once
  • Smart Separation - AI distinguishes between different action items

Best Practices

  • Include Context - Paste entire email threads or conversations for better task separation
  • Personal Tasks Only - AI identifies tasks assigned to you specifically
  • Review All Results - Check each extracted task before saving
  • Remove Duplicates - Use the √ó button to remove unwanted tasks
  • Provide Detail - More context helps AI extract and separate tasks accurately
üí° Pro Tip: When pasting email threads with multiple people and action items, the AI will extract all tasks relevant to you while ignoring tasks assigned to others. Each task appears as a separate, editable item.

üîß Troubleshooting

Authentication Issues

Can't Log In

  • Check Credentials - Verify username/email and password
  • Case Sensitivity - Usernames and passwords are case-sensitive
  • Network Connection - Ensure stable internet connection
  • Browser Cache - Clear browser cache and cookies
  • Server Status - Check if the server is running and accessible

Session Expired

  • Automatic Logout - Sessions expire after 30 days of inactivity
  • Re-login Required - Simply log in again with your credentials
  • Data Preservation - Your data is safely stored and will reload after login

AI Service Issues

Connection Failures

  • Test Connection - Use the "Test Connection" button in settings
  • Check API Keys - Verify OpenAI or Open WebUI API keys are correct
  • Server Status - Ensure Ollama or Open WebUI servers are running
  • Network Access - Check firewall and network connectivity
  • Service Limits - Verify you haven't exceeded API rate limits

Extraction Problems

  • No Tasks Found - AI might not detect tasks in the provided text
  • Poor Quality Results - Try providing more context or clearer text
  • Timeout Errors - Large texts may take longer; try smaller chunks
  • Model Issues - Try switching to a different AI model
  • Name Recognition - Update name variations in settings

Data Sync Issues

Data Not Syncing

  • Check Login Status - Ensure you're logged in to your account
  • Network Connection - Verify stable internet connection
  • Auto-Save Status - Data saves automatically every 30 seconds
  • Manual Save - Changes save when you switch tabs or close modals
  • Server Connectivity - Check if the backend server is accessible

Missing Data

  • Wrong Account - Verify you're logged into the correct account
  • Data Isolation - Each account has completely separate data
  • Recent Changes - Check if data was modified on another device
  • Backup Restore - Restore from a recent backup if available

Performance Issues

Slow Loading

  • Large Dataset - Many tasks or notes can slow performance
  • Browser Cache - Clear browser cache and reload
  • Network Speed - Check internet connection speed
  • Server Load - High server load can affect response times
  • Device Resources - Close other browser tabs or applications

Memory Issues

  • Browser Memory - Restart browser if it becomes sluggish
  • Large Notes - Very large notes can impact performance
  • Export Data - Export and clean up old completed tasks
  • Device Storage - Ensure sufficient device storage space

Import/Export Issues

Import Failures

  • File Format - Ensure file is in supported format (JSON, MD, CSV)
  • File Size - Very large files may fail to import
  • Data Format - Check that data structure matches expected format
  • Encoding Issues - Ensure file is UTF-8 encoded
  • Browser Permissions - Allow file access in browser settings

Export Problems

  • Download Blocked - Check browser download permissions
  • File Size Limits - Large exports may be blocked by browser
  • Format Issues - Try different export formats if one fails
  • Browser Compatibility - Some older browsers may have issues

Getting Additional Help

Self-Help Resources

  • Settings ‚Üí Test Connection - Diagnose AI service issues
  • Performance Report - Check browser console for performance data
  • Export Backup - Always backup data before troubleshooting
  • Browser Console - Check for error messages in developer tools

When to Seek Support

  • Persistent Login Issues - Account access problems
  • Data Loss - Missing or corrupted data
  • Server Errors - Backend connectivity issues
  • Feature Bugs - Unexpected application behavior
  • Security Concerns - Account security issues
